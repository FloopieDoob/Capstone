# Capstone
Project ini merupakan Capstone Project Module 1 yang dibuat menggunakan Python.  Aplikasi ini merupakan solusi dari pembuat berdasarkan studi kasus Aplikasi Pengolah Data Karyawan

# Tentang Aplikasi
Aplikasi ini menggunakan 2 modul yang wajib diinstal yakni:
1. prettytable
2. pyinputplus

Struktur Data:
1. ID unik (format: A001, A002, dst.)
2. Nama
3. Tempat & Tanggal Lahir
4. Jabatan
5. Bagian
